# Malla interactiva – Derecho (Universidad de Valparaíso)

Este repo publica una malla interactiva (GitHub Pages) con:
- Nombre de asignatura
- Créditos
- Requisitos
- Estado: pendiente / en-curso / aprobado (se guarda en tu navegador)

## Publicar en GitHub Pages
1. Sube `index.html` y `courses.json` a un repo.
2. Settings → Pages → Deploy from a branch → `main` + `/(root)`.
3. Abre la URL que te entrega GitHub.

## Editar ramos
Modifica `courses.json`.
